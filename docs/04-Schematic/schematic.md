# Schematic

## Overview
This schematic is a complete, gas monitoring node built around the ESP32-S2/S3.  

**Key features highlighted in the design:**
- ESP32 as central controller (Wi-Fi/BLE, ADC, UART, GPIO)
- Multi-gas sensor interface (5 V heater + 3.3 V analog output)
